# Smart Study Planner - Frontend Application

## Project Description

Frontend of Smart Study Planner built using React, TypeScript, Vite, Redux Toolkit and Tailwind CSS.

It provides a responsive UI for managing subjects, tasks, and AI-generated study plans with backend API integration.

## Technologies Used

* React
* TypeScript
* Vite
* Redux Toolkit
* Tailwind CSS
* Axios
* React Router DOM

## Features

* User Authentication (Login/Register)
* JWT Secure Access
* Dashboard View
* Subject Management (CRUD)
* Task Management (CRUD)
* AI Study Plan Integration
* Responsive UI Design

## Setup Instructions

```bash
npm install
npm run dev
```

## Environment Variables

```env
VITE_API_BASE_URL=http://localhost:5000/api/v1
```
